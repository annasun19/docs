---
title: "Update Event"
openapi: "PUT /events/{eventId}"
description: "Update an existing event's details"
---

Update the details of an existing event. You can modify any of the fields that were set during event creation.

### Path Parameters

<ParamField path="eventId" type="string" required>
  The ID of the event to update
</ParamField>

</ParamField>

<ParamField body="start_date" type="datetime" required>

<ParamField body="end_date" type="datetime" required>
